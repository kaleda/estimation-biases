# estimation-biases

This repository contains code for the paper "Conformity to popular, not average, opinions: Models, data, and evolution" by [Kaleda K. Denton](https://github.com/kaleda), Marcus W. Feldman, and [Jonathan F. Johannemann](https://github.com/JonathanJohann), published in _Proceedings of the National Academy of Sciences_, 123(25): e2530712123.
